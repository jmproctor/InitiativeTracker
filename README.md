# Project Name: Initiative Tracker

## Project Description

Initiative Tracker is an Android application designed to help Dungeon Masters and players manage combat encounters in tabletop role-playing games such as Dungeons & Dragons. The application allows users to create encounters, manage characters and NPCs, track initiative order, and monitor character health during combat.

## Project Tasks

* Set up the Android Studio development environment
* Design the user interface and application flow
* Implement user authentication
* Develop encounter and character management features
* Implement health tracking functionality
* Test and debug the application
* Create project documentation

## Project Skills Learned

* Android application development
* SQLite database management
* Activity navigation using Intents
* Custom ListView adapters
* Custom Visuals in XML (Health bars)

## Languages Used

* **Kotlin** – Application logic
* **XML** – User interface layouts
* **SQLite** – Local database storage

## Application Features

* User Registration and Login
* Encounter Creation and List
* Character Creation and List
* Initiative Tracking
* Character Details Screen
* Damage and Healing Controls
* Character Deletion
* Color changing health bars on the Character List
